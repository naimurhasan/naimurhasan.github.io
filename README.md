## Portfolio Site - Local Development

This repository also hosts my portfolio website built with Jekyll. To run it locally:

```bash
# 1. Install Ruby dependencies
bundle install

# 2. Build the site
bundle exec jekyll build

# 3. Start the development server
bundle exec jekyll serve
```

The site will be available at `http://127.0.0.1:4000` with auto-regeneration enabled.
